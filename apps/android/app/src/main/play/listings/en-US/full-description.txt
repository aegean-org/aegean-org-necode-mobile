NeCode Mobile is the Android companion for NeCode.

Use NeCode Mobile to connect back to your own computer, start or resume coding sessions, and work with local project context from your phone. The mobile app talks to your local NeCode daemon through a relay; project files and agent execution stay on your machine.

What you can do:
- Pair with your local NeCode daemon by QR code
- Start, resume, and manage project-aware coding threads
- Send text and image messages from Android
- Review markdown, code blocks, and reasoning output clearly

Built for engineers who want mobile access without moving their code into a third-party mobile service.
